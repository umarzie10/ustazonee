# UstaZone

## Project Overview

**UstaZone** is a platform to connect users with skilled professionals for home services such as plumbing, electrical work, cleaning, furniture assembly, and more. The platform makes it easy to find, book, and manage service requests quickly.

---

## Project Info

**Repository:** https://github.com/username/ustazone  
**Project Type:** Frontend (React + Vite + TypeScript)  

---

## Getting Started

You can work locally using your preferred IDE. Make sure Node.js and npm are installed.

### Steps to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/username/ustazone.git

# 2. Navigate to the project folder
cd ustazone

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
